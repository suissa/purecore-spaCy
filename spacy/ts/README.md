# spaCy core TypeScript migration

Esta pasta inicia a migração dos utilitários principais do diretório `spacy/` para TypeScript.
Atualmente inclui:

- `about.ts` com os metadados básicos da biblioteca.
- `pipe_analysis.ts` com equivalentes em TypeScript para validação de atributos e análise de pipeline.

Execute `npm install` seguido de `npm run build` dentro desta pasta para gerar os artefatos compilados em `dist/`.
